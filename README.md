# Basic instructions 

1 - Create Virtual Environment
```
python -m venv venv
```

2 - Navigate to Virtual Environment
```
cd venv\Scripts
```

3 - Activate Virtual Environment
```
.\activate
```

4 - Return to main folder (write this twice)
```
cd ..
```

5 - Install requirements
```
pip install -r requirements.txt
```

6 - Run project
```
python main.py
```
